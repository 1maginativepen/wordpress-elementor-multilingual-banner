== Installation ==
	1. Upload the entire `Multilingual-Language` folder to the `/wp-content/plugins/` directory.
	2. Activate the plugin through the 'Plugins' menu in WordPress.

== Description ==
	- This Widget gives the a banner that can used in different languages prefered by the users. you can set what image will be seen in every language and set a redirect link
	- Works like as a ads image.
